Installation¶

Python pip¶

Resistics is the Python package repository and can be installed from using pip as follows:

python -m pip install --user resistics

This is the simplest method of installing resistics and deals with all the dependencies.

Warning

resistics requires Python 3.6 or higher. Python can be downloaded from the Python downloads page and is available for all major operating systems.

Dependencies¶

Resistics uses a number of Python 3 libraries. These are:

  • numpy
  • scipy
  • pyfftw
  • configObj
  • matplotlib

These will be installed automatically when installing resistics through the Python pip package manager.

pyfftw does have an external dependency on FFTW (Fastest Fourier Transform in the West) library. More information about pyfftw can be found here: https://hgomersall.github.io/pyFFTW/
