Roadmap
-------

There is a roadmap for resistics and several potential improvements and projects are outlined below. However there are no set deadlines for accomplishing the goals. 

Areas for further work include:

- Improved robust regression
- Improved estimation of the regression uncertainty to feed into inversion
- Support for SPAM XTRX header files
- Full support for Phoenix data
- Supporting Lemi data
- Investigating how the statistics can be best combined
- Addition of signal processing methods to denoise in time domain
- Possible denoising in frequency domain
- Extraction of more meaningful window features

Feature requests or suggestions
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Feature requests and suggestions should be submitted on the `GitHub repository <https://github.com/resistics/resistics>`_ and are always welcome.
