resistics.calculators.windower module¶

class resistics.calculators.windower.Windower(refTime: datetime.datetime, timeData: resistics.dataObjects.timeData.TimeData, winSize: int, winOlap: int)[source]¶

Bases: resistics.calculators.calculator.Calculator

Class for windowing time data with overlaps

Given time data for a decimation level and a reference time, the windower calculates the number of windows and returns them.
