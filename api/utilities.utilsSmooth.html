utilities.utilsSmooth module¶

utilities.utilsSmooth.smooth1d(x: numpy.ndarray, winLen: int = 11, window: str = 'hanning')[source]¶

Smooth in 1 dimension

Parameters:
x : np.ndarray

The data array to be smoothed

winLen : int

The number of samples to smooth across

window : str

The window function

Notes

The data is padded before the smoothing to ensure that the output data is the same size as the input.

todo: The window parameter could be the window itself if an array instead of a string

Smooth in 2-D
